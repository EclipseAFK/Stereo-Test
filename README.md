# Stereo Test

Stereo Test is a command based channel testing tool made in python meant to be very simple

---

## 📦 download

You can download the official installer from this release here:

👉 https://github.com/EclipseAFK/Stereo-Test/releases/tag/frozen_release

This release is a installer and I'd reccommend just this if you aren't familiar with computers and/or such, it's a familiar installer and creates all the shortcuts for you.
